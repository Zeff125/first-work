💡 1. 함수의 개념

함수는 특정 작업을 수행하도록 설계된 독립적인 코드 블록입니다. 재사용성이 높아지고 코드가 깔끔해집니다.

🏗️ 2. 함수의 기본 구조

// 함수의 정의 (Function Definition)
int add(int a, int b) {
    int sum = a + b; // 함수 몸체 (Body)
    return sum; // 결과 반환
}

📜 3. 함수의 선언(Declaration)과 정의(Definition)

컴파일러에게 함수의 존재를 미리 알리는 것을 선언(원형, Prototype)이라고 합니다.

// 1. 함수 선언 (상단)
int add(int a, int b);

int main() {
    add(3, 5); // 호출 가능
}
// 2. 함수 정의 (하단)
int add(int a, int b) {
    return a + b;
}

🔄 4. 함수 호출과 실행 흐름

함수가 호출되면 현재 위치를 기억하고 함수의 코드로 점프합니다. 스택(Stack) 메모리에 정보가 쌓입니다.

코드 실행 흐름

1: int main() {
2:     int x = 10;
3:     int result = func(x); // 호출!
4:     printf("%d", result);
5:     return 0;
6: }

7: int func(int n) {
8:     return n * 2;
9: }
